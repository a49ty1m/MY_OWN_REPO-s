## Phase 1: De-Cluttering

Remove distractions from the default Brave setup.

### Dashboard Settings
Open a new tab → click **Gear Icon** [[00:22](http://www.youtube.com/watch?v=tSfDZiK3eHk&t=22)]

| Setting | Action |
|---------|--------|
| Background | Turn off "Sponsored Images" |
| Widgets | Disable "Top Sites," "Brave News," "Clock," "Cards" |
| Search Bar | Disable (URL bar already handles search) [[00:44](http://www.youtube.com/watch?v=tSfDZiK3eHk&t=44)] |

### Interface Cleanup
- **Top Bar Icons:** Right-click and hide unused icons (Rewards, Wallet, etc.) [[00:49](http://www.youtube.com/watch?v=tSfDZiK3eHk&t=49)]
- **Sidebar:** Right-click → set to **"Never"** [[00:54](http://www.youtube.com/watch?v=tSfDZiK3eHk&t=54)]

## Phase 3: Extensions

### 1. Catppuccin Theme
Install **Catppuccin** (Mocha version recommended) [[01:28](http://www.youtube.com/watch?v=tSfDZiK3eHk&t=88)]
- Provides a cohesive, professional color palette

### 2. Dark Reader
Advanced dark mode for all websites [[01:55](http://www.youtube.com/watch?v=tSfDZiK3eHk&t=115)]

**Setup:**
1. Go to **More > All Settings > Advanced > Dev Tools**
2. Enable **Design Prototype** [[02:02](http://www.youtube.com/watch?v=tSfDZiK3eHk&t=122)]
3. Paste custom hex codes matching your Catppuccin theme [[02:13](http://www.youtube.com/watch?v=tSfDZiK3eHk&t=133)]

### 3. Stylus
Custom CSS injection for specific sites [[02:42](http://www.youtube.com/watch?v=tSfDZiK3eHk&t=162)]

**Setup:**
1. Install Stylus extension
2. Click the Stylus icon → **Manage**
3. Click **Import** (backup icon) → select [stylus-catppuccin.json](stylus-catppuccin.json)
